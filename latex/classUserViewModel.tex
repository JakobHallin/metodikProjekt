\doxysubsection{Detailed Description}
View\+Model class that connects the \mbox{\hyperlink{classUser}{User}} model to the View interface. 

Provides access to user data and user-\/related account View\+Models. It serves as an intermediary between the \mbox{\hyperlink{classUser}{User}} model and the view layer in an MVVM architecture.

\doxysubsection{Detailed Description}
Provides simplified access to executing and preparing SQL queries using Qt\textquotesingle{}s SQL module. 

The \mbox{\hyperlink{classSql}{Sql}} class offers methods for executing raw SQL, preparing parameterized statements, binding values generically, and performing templated inserts. It uses the connection from \mbox{\hyperlink{classDb}{Db}}.

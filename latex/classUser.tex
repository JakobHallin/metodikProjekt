\doxysubsection{Detailed Description}
Manages a collection of accounts and stock operations for a user. 

The \mbox{\hyperlink{classUser}{User}} class handles multiple \mbox{\hyperlink{classAccount}{Account}} instances, identified by a unique user\+ID. It provides functionality for buying/selling stocks across accounts, checking balances, and verifying if a particular stock exists.

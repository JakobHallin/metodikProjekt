\doxysubsection{Detailed Description}
Main window for displaying and interacting with a User’s multiple accounts. 

The userview class creates a scrollable list of accountview widgets, one per account owned by the user. It updates whenever the \mbox{\hyperlink{classUserViewModel}{User\+View\+Model}} provides new accounts.

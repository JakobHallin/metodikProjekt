\doxysubsection{Detailed Description}
QWidget-\/based view class for displaying stock information. 

The stockview class displays the stock ID and current amount, and updates when the underlying \mbox{\hyperlink{classStockViewModel}{Stock\+View\+Model}} emits amount\+Changed().

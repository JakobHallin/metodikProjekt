\doxysubsection{Detailed Description}
View\+Model class that wraps an \mbox{\hyperlink{classAccount}{Account}} object for interaction with the UI. 

Provides observable access to the account’s ID, balance, and its stocks’ view models. Facilitates buy/sell operations by forwarding to the \mbox{\hyperlink{classAccount}{Account}} model.
